# 信息流基础入门

## 01-解析信息流素材 流量类型和算法逻辑

### 电商常见流量模型

货架电商，也叫做搜索电商，是人找货的模式，像淘宝，京东，拼多多等传统电商，现在的抖音商城都是以人找货为主的，也被称之为一类电商，人找货是属于搜索流量的，也叫做意向性流量，来的都是精准流量，因为都是用户主动发起的搜索行为

直播电商，也叫做兴趣电商，它是货找人，系统用算法把产品主动的推送给可能购买的人或者可能感兴趣的人，用户是被动的接受推送信息，比如说直播引流视频，小黄车带货视频，都是属于货找人的那种，是属于信息流广告，也叫做二类电商

### 两种流量模型的区别

一个是用户有明确的购物需求，主动地去寻找某个特定的产品，目标明确，意向强烈，另一个是漫无目的的瞎晃悠，看热闹
一类电商的核心是选关键词，关键词越精准，卖家的需求越明确，理论上转化率就会越高
二类电商是选关键词背后的人群，只有精准的人群没有精准的关键词，因为同一个卖家会有多个不同类目的产品同时感兴趣，例如对瑜伽裤，小白鞋这两个产品同时感兴趣的人群，基本就是同一批人。

做兴趣电商的内容，也就是信息流编导，一定要注意细节，细节决定成败。通过短视频进入你直播间的人，他们的年龄，性别， 兴趣爱好，基本上都是由你的短视频素材决定的。你只做什么样子的素材，就会吸引什么样子的人进入到你的直播间
制作喜欢健身的小姐姐的素材，吸引的就是健身小姐姐人群，和健身小姐姐无关的人群，看到这个短视频素材就会直接划走。

### 信息流算法逻辑

#### 推流机制

大部分信息流平台（抖音，快手，小红书），都是基于算法推荐机制，遵循“冷启动=小流量测试=正反馈=放量=衰退期”的流程

#### 关键指标

平台推荐的核心是素材的表现数据，主要包括：

点击率CTR：是否有吸引人点击的标题，封面，开头，在信息流里面，点击率是分两种的，一种是视频点击率，计算公式为 **素材点击次数/素材曝光*100%** 视频点击人数具体是 **按照点击行为来计算的（点击封面，点击评论区互动，点昵称，点头像，点商品卡等所有可点击区域的总和）** 另一种是点击进入率，计算公式为 **商品落地页/点击进入直播次数/素材曝光人数 * 100%** 具体是按照**点击商品落地页/点击进入直播间次数来统计的**

完播率 3s/5s/整体 用户是否愿意看完，如果素材都看不完，那大概率是不会进入直播间的

转化率CVR 用户是否愿意下单购买或进入落地页

互动率 是否有点赞，评论，转发

#### 信息流推荐流程

1. 初始 冷启动 视频发布过审核/审核建议/审核失败
2. 首轮 小范围测试 系统推荐给500~1000人小样本测试
3. 二轮 若各项指标合格，扩大流量
4. 三轮以上 持续放量直到表现下滑或者素材疲劳
5. 若首轮点击率，完播率，转化率不达标，即刻断流，也就是不消耗

### 信息流编导五大类型

信息流短视频编导，听着简单，其实根据不同的项目目标，编导做的事情和考核目标都不太一样，但大概可以分为五类：

#### 挂车短视频

视频下面挂一个购物车链接，目的是让用户可以快速地下单购买，挂车类短视频其实是比第二种短视频引流视频更难做的，因为短视频引流视频塑品还有直播间做承接，有主播负责二次情绪触达，去做转化行为，而**挂车短视频却要求用户，在看完视频第一瞬间做出决策行为下单**，难度可想而知

#### 引流直播间短视频

千川编导做的短引视频，不直接成交，而是让用户心甘情愿地点进直播间，由直播间做承接转化



#### 推广App短视频

有些公司是做工具类app或者小游戏app，这类信息编导就是需要让用户觉得下载这个app能够解决自己的痛点，或者能给我带来快感，满足自己的需求，通常用情景小剧场引起兴趣，结尾引导点击下载。



#### 线索留资短视频

涉及到一些教育，招商加盟，培训等行业，这类短视频大部分是先做用户兴趣教育的，勾起他们想了解更多的欲望，结尾引导他们留下联系方式（比如说加企业微信，添加表单）



#### CID短视频

跨平台投放，**引流到其他电商平台进行成交**，比如有些品牌方不是在抖音卖，而是想把人引流到天猫，京东，拼多多去下单，短视频一般就起到种草作用，让用户觉得产品靠谱，然后再顺势引导用户到这些外部平台去买

不同类型的编导工作，逻辑和落地执行都不太一样，

1. 做挂车短视频，就得冲击力和卖点够强
2. 做引流直播，核心是留人
3. 做app下载，得让用户觉得实在有趣
4. 做留资，就是要讲故事，铺梗伏笔
5. 做外部引流，就要埋下购买欲望



#### 常见行业话术

| 术语     | 中文名                                   | 解释                                                         | 白话                                                        | 分析                                                         |
| -------- | ---------------------------------------- | ------------------------------------------------------------ | ----------------------------------------------------------- | ------------------------------------------------------------ |
| CTR      | 点击率                                   | 点击次数/展示次数×100%                                       | 视频/封面被刷到后，有多少人愿意点进来看                     | 太低说明封面标题不够吸引人，该换狗子或者文案结构             |
| CVR      | 转化率                                   | 成交次数/点击次数×100%                                       | 点进来的人有多少最后下单或做了你想让他做的动作（加购/留资） | 素材有比较强的说服力才有高CVR                                |
| GMV      | 总销售额                                 | 商品交易总额，不扣除退换货                                   | 视频/直播间总共卖了多少钱                                   | 衡量你带货的体量                                             |
| NMV      | 退货后总销售额                           | GMV-退款/取消等                                              | 扣掉退货，退款后真正卖出去的金额                            | 更真实的反应赚钱能力，方便更好的算出实际盈利                 |
| ROI      | 投入产出比                               | 收益/成本                                                    | 花了多少钱，赚了多少钱，大于1赚钱，小于1亏钱                | 编导策划内容大部分都要服务于这个指标                         |
| CPM      | 千次展现成本                             | 每曝光1000次展现花费费用                                     | 平台给你曝光1000次，你要花多少钱去买                        | 适合我们来观测爆款素材的跑量成本                             |
| CPC      | 每次点击成本                             | 每次点击花费费用                                             | 每次点一下视频/广告，需要支付的费用                         | 目标客户越精准，越吸引人，CPC越低                            |
| OCPM     | 优化目标千次展示出价，按目标转化优化展示 | 设定一个目标，比如成交，系统会把曝光优先分配给更容易成交的人群 | /                                                           | /                                                            |
| ECPM     | 有效每千次展示收入                       | 每1000次曝光能收回来的费用，也就是每1000次曝光的销售额       | 通过1000次曝光能赚到多少钱（计算公式：出价×CTR×CVR×1000）   | 不同广告主同一时间赛马，谁的ECPM排名高，谁就能优先获取流量   |
| OCPC     | 优化点击出价，按转化质量优化点击         | 和普通的CPC不同，系统不仅看点击，还看点击之后是否有转化，投放更加精准 | /                                                           | /                                                            |
| sku      | 商品细分链接                             | 商品最小存货单位                                             | 一个SKU=一个商品的具体型号，比如白色M码T恤                  | 素材内要写商品链接里有的规格，没有的不要写，不然就是虚假宣传 |
| UV       | 独立访客数                               | 一天内同一个人只记录一次                                     | 有多少不同的人，看过这条内容，不重复计算                    | UV越高说明内容覆盖量越广                                     |
| UV价值   | 每个独立访客带来的平均收益               | 独立访客平均收益=总收益/UV                                   | 每一个来看你视频的人，平均帮你赚了多少钱                    | 一般情况下UV价值越高，说明人群越精准                         |
| PV       | 页面浏览总数                             | 页面或者视频总共被刷了多少次                                 | 同一个人看10次算10次                                        | /                                                            |
| PV价值   | 每次页面浏览带来的收益                   | 每次浏览平均收益=总收益/pv                                   | 和UV价值类似，更看重用户行为活跃度，如是否停留，点链接      | /                                                            |
| 蓝海     | 小众刚需赛道                             | 市场竞争小潜力大，玩家少                                     | 竞争小，内容供给少，用户需求大的领域                        | /                                                            |
| 红海     | 都在卷的类目                             | 高度竞争，内容供大于求的市场                                 | 美妆，护肤，穿搭                                            | /                                                            |
| 标品     | 标准化产品                               | 功能清晰，价格透明，用户决策成本低的产品                     | 充电宝，灯泡这类产品，卖家看重性价比                        | 内容侧重实用/价格优势                                        |
| 非标品   | 非标准化产品                             | 主观审美影响大，价格跨度大，体验差异大的产品                 | 衣服，饰品，家居摆件，需要依靠氛围感，审美打动吸引用户的    | 内容侧重风格和情绪价值                                       |
| 黑五类   | 广告平台限制或者禁止投放的产品类别       | 减肥，丰胸，保健，虚假功效类产品                             | 利润高，且没有资质的功效类产品，都是黑五类                  | 作为编导能不碰就不碰                                         |
| aware    | A1人群                                   | 被你的产品，直播间，商品卡，短视频首次曝光过的人             | 知道你的人                                                  | 知道                                                         |
| appeal   | A2人群                                   | 在A1人群基础上，点击过你的产品，视频评论区，进过你的直播间，看过你短视频的人 | 喜欢你的人                                                  | 喜欢                                                         |
| ask      | A3人群                                   | 在A2人群基础上，架构了你的产品，进直播间和短视频与你互动过   | 相信你的人，种草和收割就是这拨人                            | 相信                                                         |
| act      | A4人群                                   | 首次购买                                                     | 第一次购买过你产品的人                                      | 下单                                                         |
